QUICK START
══════════════════════════════════════════════════════════
1. cd ~/Desktop/GO/GoMaster
2. ./validate-project.sh          # Verify everything is OK
3. open -a Xcode .                 # Open in Xcode
4. Configure signing
5. ⌘ + R to run

BUILD FOR APP STORE
══════════════════════════════════════════════════════════
1. ./build-for-appstore.sh         # Automated build
2. Product → Archive in Xcode
3. Distribute to App Store Connect
4. Complete metadata (see AppStore/AppStoreMetadata.md)
5. Submit for review
